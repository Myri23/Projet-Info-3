Commande utilile:
--

| Option | Description                                            |
|--------|--------------------------------------------------------|
| -d1    | Conducteurs avec le plus grand nombre de trajets.      |

| -t     | Les 10 villes les plus traversées.                    |
| -s     | Statistiques sur les étapes.                          |
| -h     | Pour afficher l'aide.                                 |
